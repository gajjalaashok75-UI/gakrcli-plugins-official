# Use Marketing Ideas

Use this command when you want GAKRCLI to apply the `marketing-ideas` skill.

## Usage

```text
/marketing-ideas:use
```

## What It Does

- Loads the `marketing-ideas` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `marketing-ideas-agent` when an agent is useful.
