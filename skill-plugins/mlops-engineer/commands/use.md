# Use Mlops Engineer

Use this command when you want GAKRCLI to apply the `mlops-engineer` skill.

## Usage

```text
/mlops-engineer:use
```

## What It Does

- Loads the `mlops-engineer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `mlops-engineer-agent` when an agent is useful.
