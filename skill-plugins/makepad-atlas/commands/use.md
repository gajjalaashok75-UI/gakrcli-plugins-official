# Use Makepad Atlas

Use this command when you want GAKRCLI to apply the `makepad-atlas` skill.

## Usage

```text
/makepad-atlas:use
```

## What It Does

- Loads the `makepad-atlas` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `makepad-atlas-agent` when an agent is useful.
