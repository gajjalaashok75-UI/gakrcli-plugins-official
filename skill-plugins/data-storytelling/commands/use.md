# Use Data Storytelling

Use this command when you want GAKRCLI to apply the `data-storytelling` skill.

## Usage

```text
/data-storytelling:use
```

## What It Does

- Loads the `data-storytelling` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `data-storytelling-agent` when an agent is useful.
