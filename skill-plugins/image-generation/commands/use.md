# Use Image Generation

Use this command when you want GAKRCLI to apply the `image-generation` skill.

## Usage

```text
/image-generation:use
```

## What It Does

- Loads the `image-generation` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `image-generation-agent` when an agent is useful.
