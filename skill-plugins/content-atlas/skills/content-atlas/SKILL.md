## PRIMACY ZONE — Identity and Purpose

**What this atlas contains**

Content-atlas is your comprehensive guide for content creation and strategy. It orchestrates specialized skills covering content strategy, creation, marketing, social media content, and content optimization.

**Use this atlas when**

- Creating content (articles, blog posts, social media)
- Developing content strategy
- Planning content marketing campaigns
- Optimizing content for engagement
- Creating social media content
- Managing content workflows

**Do not use this atlas when**

- Technical documentation (use docs-architect, documentation)
- UI/UX copywriting (use frontend-atlas)
- Code documentation (use code-reviewer)
- Product documentation (use product-atlas)

### Data Extraction for Content Research

Use **data-extraction** skill to gather content ideas, trends, and research data:

| Platform Category | Platforms | Content Use Cases |
|-------------------|-----------|-------------------|
| **Social Media** | Reddit, Medium, Quora | Trending topics, user discussions, content ideas |
| **Developer Platforms** | GitHub, Stack Overflow, Dev.to | Technical content, code examples, developer trends |
| **Entertainment** | Spotify, Letterboxd, Goodreads | Cultural trends, reviews, recommendations |
| **Academic** | arXiv, PubMed, Crossref | Research-backed content, citations, academic insights |
| **Business** | ProductHunt, G2, Capterra | Product trends, user reviews, market insights |

**Example workflows:**
- Blog post research: `data-extraction` (Reddit, Medium) → `content-creator` (write) → `context-optimization` (SEO)
- Tech article: `data-extraction` (GitHub, Stack Overflow) → `article-writing` (long-form) → `humanizer` (relatable)
- Product review: `data-extraction` (G2, Capterra) → `content-marketer` (marketing angle) → `image-generation` (visuals)
- Trend analysis: `data-extraction` (social media) → `data-storytelling` (narrative) → `chart-visualization` (visuals)

- content-creator (general content creation)

**When to invoke external skills:**

- Research → deep-research, market-research

- Social campaign: social-content + content-marketer + image-generation
- Email sequence: email-sequence + marketing-psychology + humanizer

**Content workflow:**
1. Research (deep-research, market-research)
2. Strategy (content-strategy)
3. Creation (content-creator, article-writing)
4. Optimization (context-optimization)
5. Distribution (social-content, email-sequence)

**Remember:** Content-atlas focuses on marketing and creative content. For technical documentation, use docs-architect. For product documentation, use product-atlas. For UI copy, use frontend-atlas.
