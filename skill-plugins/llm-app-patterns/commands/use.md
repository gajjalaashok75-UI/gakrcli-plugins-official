# Use LLM App Patterns

Use this command when you want GAKRCLI to apply the `llm-app-patterns` skill.

## Usage

```text
/llm-app-patterns:use
```

## What It Does

- Loads the `llm-app-patterns` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `llm-app-patterns-agent` when an agent is useful.
