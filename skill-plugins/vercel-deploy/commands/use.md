# Use Vercel Deploy

Use this command when you want GAKRCLI to apply the `vercel-deploy` skill.

## Usage

```text
/vercel-deploy:use
```

## What It Does

- Loads the `vercel-deploy` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `vercel-deploy-agent` when an agent is useful.
