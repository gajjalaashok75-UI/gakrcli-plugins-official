# Use Email Systems

Use this command when you want GAKRCLI to apply the `email-systems` skill.

## Usage

```text
/email-systems:use
```

## What It Does

- Loads the `email-systems` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `email-systems-agent` when an agent is useful.
