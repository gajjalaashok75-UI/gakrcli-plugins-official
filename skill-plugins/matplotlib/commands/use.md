# Use Matplotlib

Use this command when you want GAKRCLI to apply the `matplotlib` skill.

## Usage

```text
/matplotlib:use
```

## What It Does

- Loads the `matplotlib` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `matplotlib-agent` when an agent is useful.
