# Use Landing Page Generator

Use this command when you want GAKRCLI to apply the `landing-page-generator` skill.

## Usage

```text
/landing-page-generator:use
```

## What It Does

- Loads the `landing-page-generator` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `landing-page-generator-agent` when an agent is useful.
