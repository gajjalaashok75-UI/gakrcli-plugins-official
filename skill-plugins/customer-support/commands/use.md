# Use Customer Support

Use this command when you want GAKRCLI to apply the `customer-support` skill.

## Usage

```text
/customer-support:use
```

## What It Does

- Loads the `customer-support` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `customer-support-agent` when an agent is useful.
