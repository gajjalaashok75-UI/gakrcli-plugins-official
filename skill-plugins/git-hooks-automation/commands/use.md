# Use Git Hooks Automation

Use this command when you want GAKRCLI to apply the `git-hooks-automation` skill.

## Usage

```text
/git-hooks-automation:use
```

## What It Does

- Loads the `git-hooks-automation` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `git-hooks-automation-agent` when an agent is useful.
