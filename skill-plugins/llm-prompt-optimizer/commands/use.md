# Use LLM Prompt Optimizer

Use this command when you want GAKRCLI to apply the `llm-prompt-optimizer` skill.

## Usage

```text
/llm-prompt-optimizer:use
```

## What It Does

- Loads the `llm-prompt-optimizer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `llm-prompt-optimizer-agent` when an agent is useful.
