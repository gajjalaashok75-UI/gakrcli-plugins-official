# Use Langgraph

Use this command when you want GAKRCLI to apply the `langgraph` skill.

## Usage

```text
/langgraph:use
```

## What It Does

- Loads the `langgraph` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `langgraph-agent` when an agent is useful.
