# Use Podcast Generation

Use this command when you want GAKRCLI to apply the `podcast-generation` skill.

## Usage

```text
/podcast-generation:use
```

## What It Does

- Loads the `podcast-generation` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `podcast-generation-agent` when an agent is useful.
