# Use Threejs Atlas

Use this command when you want GAKRCLI to apply the `threejs-atlas` skill.

## Usage

```text
/threejs-atlas:use
```

## What It Does

- Loads the `threejs-atlas` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `threejs-atlas-agent` when an agent is useful.
