## PRIMACY ZONE — Identity and Purpose

**What this atlas contains**

Threejs-atlas orchestrates Three.js and WebGL development skills for creating 3D web experiences.

**Use this atlas when**

- Building 3D web applications
- Creating WebGL graphics
- Implementing 3D animations
- Working with 3D models and textures
- Creating custom shaders
- Building interactive 3D experiences

**Do not use this atlas when**

- 2D graphics (use canvas-design in frontend-atlas)
- UI components (use frontend-atlas)
- Backend 3D processing (use backend-atlas)

## Usage Patterns

**Building a 3D scene:**
1. threejs-fundamentals (setup)
2. threejs-geometry (create objects)
3. threejs-materials (apply materials)
4. threejs-lighting (add lights)
5. threejs-animation (animate)
6. threejs-interaction (add controls)

**Loading 3D models:**
1. threejs-loaders (load models)
2. threejs-materials (adjust materials)
3. threejs-lighting (lighting setup)
4. threejs-animation (animations)

**Custom shaders:**
1. threejs-fundamentals (setup)
2. threejs-shaders (write shaders)
3. threejs-materials (shader materials)

**React integration:**
1. react-atlas (React setup)
2. threejs-fundamentals (Three.js basics)
3. threejs-* (specific features)
