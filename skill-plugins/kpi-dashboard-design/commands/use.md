# Use KPI Dashboard Design

Use this command when you want GAKRCLI to apply the `kpi-dashboard-design` skill.

## Usage

```text
/kpi-dashboard-design:use
```

## What It Does

- Loads the `kpi-dashboard-design` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `kpi-dashboard-design-agent` when an agent is useful.
