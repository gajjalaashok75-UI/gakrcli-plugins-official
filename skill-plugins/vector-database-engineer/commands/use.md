# Use Vector Database Engineer

Use this command when you want GAKRCLI to apply the `vector-database-engineer` skill.

## Usage

```text
/vector-database-engineer:use
```

## What It Does

- Loads the `vector-database-engineer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `vector-database-engineer-agent` when an agent is useful.
