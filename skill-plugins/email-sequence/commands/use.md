# Use Email Sequence

Use this command when you want GAKRCLI to apply the `email-sequence` skill.

## Usage

```text
/email-sequence:use
```

## What It Does

- Loads the `email-sequence` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `email-sequence-agent` when an agent is useful.
