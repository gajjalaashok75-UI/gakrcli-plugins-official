# Use Chart Visualization

Use this command when you want GAKRCLI to apply the `chart-visualization` skill.

## Usage

```text
/chart-visualization:use
```

## What It Does

- Loads the `chart-visualization` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `chart-visualization-agent` when an agent is useful.
