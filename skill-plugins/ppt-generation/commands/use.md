# Use PPT Generation

Use this command when you want GAKRCLI to apply the `ppt-generation` skill.

## Usage

```text
/ppt-generation:use
```

## What It Does

- Loads the `ppt-generation` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `ppt-generation-agent` when an agent is useful.
