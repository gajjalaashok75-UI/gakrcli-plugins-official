# Use Product Atlas

Use this command when you want GAKRCLI to apply the `product-atlas` skill.

## Usage

```text
/product-atlas:use
```

## What It Does

- Loads the `product-atlas` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `product-atlas-agent` when an agent is useful.
