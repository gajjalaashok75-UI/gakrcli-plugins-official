# Use Plotly

Use this command when you want GAKRCLI to apply the `plotly` skill.

## Usage

```text
/plotly:use
```

## What It Does

- Loads the `plotly` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `plotly-agent` when an agent is useful.
