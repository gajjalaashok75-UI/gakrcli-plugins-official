# Use Marketing Psychology

Use this command when you want GAKRCLI to apply the `marketing-psychology` skill.

## Usage

```text
/marketing-psychology:use
```

## What It Does

- Loads the `marketing-psychology` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `marketing-psychology-agent` when an agent is useful.
