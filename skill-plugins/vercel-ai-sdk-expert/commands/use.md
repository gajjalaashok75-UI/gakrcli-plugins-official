# Use Vercel AI Sdk Expert

Use this command when you want GAKRCLI to apply the `vercel-ai-sdk-expert` skill.

## Usage

```text
/vercel-ai-sdk-expert:use
```

## What It Does

- Loads the `vercel-ai-sdk-expert` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `vercel-ai-sdk-expert-agent` when an agent is useful.
