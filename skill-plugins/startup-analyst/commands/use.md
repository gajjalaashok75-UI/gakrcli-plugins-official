# Use Startup Analyst

Use this command when you want GAKRCLI to apply the `startup-analyst` skill.

## Usage

```text
/startup-analyst:use
```

## What It Does

- Loads the `startup-analyst` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `startup-analyst-agent` when an agent is useful.
