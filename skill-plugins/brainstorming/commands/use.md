# Use Brainstorming

Use this command when you want GAKRCLI to apply the `brainstorming` skill.

## Usage

```text
/brainstorming:use
```

## What It Does

- Loads the `brainstorming` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `brainstorming-agent` when an agent is useful.
