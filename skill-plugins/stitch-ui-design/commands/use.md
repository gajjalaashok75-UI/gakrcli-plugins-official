# Use Stitch UI Design

Use this command when you want GAKRCLI to apply the `stitch-ui-design` skill.

## Usage

```text
/stitch-ui-design:use
```

## What It Does

- Loads the `stitch-ui-design` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `stitch-ui-design-agent` when an agent is useful.
