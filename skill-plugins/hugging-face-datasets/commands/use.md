# Use Hugging Face Datasets

Use this command when you want GAKRCLI to apply the `hugging-face-datasets` skill.

## Usage

```text
/hugging-face-datasets:use
```

## What It Does

- Loads the `hugging-face-datasets` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `hugging-face-datasets-agent` when an agent is useful.
