# Use Ethical Hacking Methodology

Use this command when you want GAKRCLI to apply the `ethical-hacking-methodology` skill.

## Usage

```text
/ethical-hacking-methodology:use
```

## What It Does

- Loads the `ethical-hacking-methodology` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `ethical-hacking-methodology-agent` when an agent is useful.
