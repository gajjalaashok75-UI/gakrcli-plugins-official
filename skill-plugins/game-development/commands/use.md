# Use Game Development

Use this command when you want GAKRCLI to apply the `game-development` skill.

## Usage

```text
/game-development:use
```

## What It Does

- Loads the `game-development` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `game-development-agent` when an agent is useful.
