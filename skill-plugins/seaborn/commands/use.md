# Use Seaborn

Use this command when you want GAKRCLI to apply the `seaborn` skill.

## Usage

```text
/seaborn:use
```

## What It Does

- Loads the `seaborn` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `seaborn-agent` when an agent is useful.
