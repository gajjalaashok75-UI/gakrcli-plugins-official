# Use Vercel Automation

Use this command when you want GAKRCLI to apply the `vercel-automation` skill.

## Usage

```text
/vercel-automation:use
```

## What It Does

- Loads the `vercel-automation` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `vercel-automation-agent` when an agent is useful.
