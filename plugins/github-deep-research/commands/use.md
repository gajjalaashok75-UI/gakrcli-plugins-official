# Use Github Deep Research

Use this command when you want GAKRCLI to apply the `github-deep-research` skill.

## Usage

```text
/github-deep-research:use
```

## What It Does

- Loads the `github-deep-research` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `github-deep-research-agent` when an agent is useful.
