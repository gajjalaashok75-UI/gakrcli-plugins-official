# Use Hugging Face Cli

Use this command when you want GAKRCLI to apply the `hugging-face-cli` skill.

## Usage

```text
/hugging-face-cli:use
```

## What It Does

- Loads the `hugging-face-cli` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `hugging-face-cli-agent` when an agent is useful.
