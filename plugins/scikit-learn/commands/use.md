# Use Scikit Learn

Use this command when you want GAKRCLI to apply the `scikit-learn` skill.

## Usage

```text
/scikit-learn:use
```

## What It Does

- Loads the `scikit-learn` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `scikit-learn-agent` when an agent is useful.
