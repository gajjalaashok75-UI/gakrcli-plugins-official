# Use Data Scientist

Use this command when you want GAKRCLI to apply the `data-scientist` skill.

## Usage

```text
/data-scientist:use
```

## What It Does

- Loads the `data-scientist` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `data-scientist-agent` when an agent is useful.
