# Use Pytorch Patterns

Use this command when you want GAKRCLI to apply the `pytorch-patterns` skill.

## Usage

```text
/pytorch-patterns:use
```

## What It Does

- Loads the `pytorch-patterns` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `pytorch-patterns-agent` when an agent is useful.
