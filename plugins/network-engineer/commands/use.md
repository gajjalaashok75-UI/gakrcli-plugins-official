# Use Network Engineer

Use this command when you want GAKRCLI to apply the `network-engineer` skill.

## Usage

```text
/network-engineer:use
```

## What It Does

- Loads the `network-engineer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `network-engineer-agent` when an agent is useful.
