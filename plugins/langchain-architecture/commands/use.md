# Use Langchain Architecture

Use this command when you want GAKRCLI to apply the `langchain-architecture` skill.

## Usage

```text
/langchain-architecture:use
```

## What It Does

- Loads the `langchain-architecture` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `langchain-architecture-agent` when an agent is useful.
