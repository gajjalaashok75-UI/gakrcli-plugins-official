# Use Mermaid Expert

Use this command when you want GAKRCLI to apply the `mermaid-expert` skill.

## Usage

```text
/mermaid-expert:use
```

## What It Does

- Loads the `mermaid-expert` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `mermaid-expert-agent` when an agent is useful.
