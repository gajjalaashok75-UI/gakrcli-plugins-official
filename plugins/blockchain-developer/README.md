# Blockchain Developer Plugin

Build production-ready Web3 applications, smart contracts, and decentralized systems. Implements DeFi protocols, NFT platforms, DAOs, and enterprise blockchain integrations.

## Overview

This plugin packages the `blockchain-developer` skill from the souls library as a first-class GAKRCLI marketplace plugin.

## Installation

```bash
/plugin install blockchain-developer@gakrcli-plugins-official
```

## Usage

### Command

```text
/blockchain-developer:use
```

### Agent

Ask GAKRCLI to launch `blockchain-developer-agent` when you want focused help with this skill's workflow.

### Skill

The skill is available at:

```text
skills/blockchain-developer/SKILL.md
```

## Plugin Structure

```text
blockchain-developer/
|-- .gakrcli-plugin/plugin.json
|-- commands/use.md
|-- agents/blockchain-developer-agent.md
|-- skills/blockchain-developer/SKILL.md
|-- README.md
`-- LICENSE
```

Some skills include additional references, scripts, assets, or nested skill files under `skills/blockchain-developer/`.
