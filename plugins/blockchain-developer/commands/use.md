# Use Blockchain Developer

Use this command when you want GAKRCLI to apply the `blockchain-developer` skill.

## Usage

```text
/blockchain-developer:use
```

## What It Does

- Loads the `blockchain-developer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `blockchain-developer-agent` when an agent is useful.
