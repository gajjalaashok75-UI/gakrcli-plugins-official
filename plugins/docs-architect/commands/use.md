# Use Docs Architect

Use this command when you want GAKRCLI to apply the `docs-architect` skill.

## Usage

```text
/docs-architect:use
```

## What It Does

- Loads the `docs-architect` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `docs-architect-agent` when an agent is useful.
