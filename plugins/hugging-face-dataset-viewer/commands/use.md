# Use Hugging Face Dataset Viewer

Use this command when you want GAKRCLI to apply the `hugging-face-dataset-viewer` skill.

## Usage

```text
/hugging-face-dataset-viewer:use
```

## What It Does

- Loads the `hugging-face-dataset-viewer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `hugging-face-dataset-viewer-agent` when an agent is useful.
