# Use Content Atlas

Use this command when you want GAKRCLI to apply the `content-atlas` skill.

## Usage

```text
/content-atlas:use
```

## What It Does

- Loads the `content-atlas` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `content-atlas-agent` when an agent is useful.
