# Use Data Engineer

Use this command when you want GAKRCLI to apply the `data-engineer` skill.

## Usage

```text
/data-engineer:use
```

## What It Does

- Loads the `data-engineer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `data-engineer-agent` when an agent is useful.
