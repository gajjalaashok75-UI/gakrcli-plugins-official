# Use Market Research

Use this command when you want GAKRCLI to apply the `market-research` skill.

## Usage

```text
/market-research:use
```

## What It Does

- Loads the `market-research` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `market-research-agent` when an agent is useful.
