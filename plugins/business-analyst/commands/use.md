# Use Business Analyst

Use this command when you want GAKRCLI to apply the `business-analyst` skill.

## Usage

```text
/business-analyst:use
```

## What It Does

- Loads the `business-analyst` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `business-analyst-agent` when an agent is useful.
