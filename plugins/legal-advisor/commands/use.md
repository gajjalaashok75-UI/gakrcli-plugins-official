# Use Legal Advisor

Use this command when you want GAKRCLI to apply the `legal-advisor` skill.

## Usage

```text
/legal-advisor:use
```

## What It Does

- Loads the `legal-advisor` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `legal-advisor-agent` when an agent is useful.
