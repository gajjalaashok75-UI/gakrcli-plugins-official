# Use Rust Pro

Use this command when you want GAKRCLI to apply the `rust-pro` skill.

## Usage

```text
/rust-pro:use
```

## What It Does

- Loads the `rust-pro` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `rust-pro-agent` when an agent is useful.
