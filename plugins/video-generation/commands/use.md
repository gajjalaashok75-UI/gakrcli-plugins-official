# Use Video Generation

Use this command when you want GAKRCLI to apply the `video-generation` skill.

## Usage

```text
/video-generation:use
```

## What It Does

- Loads the `video-generation` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `video-generation-agent` when an agent is useful.
