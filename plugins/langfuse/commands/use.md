# Use Langfuse

Use this command when you want GAKRCLI to apply the `langfuse` skill.

## Usage

```text
/langfuse:use
```

## What It Does

- Loads the `langfuse` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `langfuse-agent` when an agent is useful.
