# Use AI Engineer

Use this command when you want GAKRCLI to apply the `ai-engineer` skill.

## Usage

```text
/ai-engineer:use
```

## What It Does

- Loads the `ai-engineer` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `ai-engineer-agent` when an agent is useful.
