# Use ML Pipeline Workflow

Use this command when you want GAKRCLI to apply the `ml-pipeline-workflow` skill.

## Usage

```text
/ml-pipeline-workflow:use
```

## What It Does

- Loads the `ml-pipeline-workflow` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `ml-pipeline-workflow-agent` when an agent is useful.
