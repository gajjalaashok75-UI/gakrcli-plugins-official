## What lives in the topic files

- Official page URL and direct links to the relevant official child pages.
- Section and subsection coverage for the corresponding docs page.
- Parameters, returns, settings, methods, properties, and callback notes where the official docs expose them.
- Fresh JS/HTML/CSS examples written for this atlas instead of copied site demos.
- Notes about official docs quirks when the source pages are internally inconsistent or omit defaults.

## Practical use

- Open this file first when routing a user request into the correct Anime.js module.
- Open the topic file next when you need the deeper section and subsection links.
- Prefer the local topic file for structured guidance, and the official page link when exact source provenance matters.
