# Use Animejs Atlas

Use this command when you want GAKRCLI to apply the `animejs-atlas` skill.

## Usage

```text
/animejs-atlas:use
```

## What It Does

- Loads the `animejs-atlas` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `animejs-atlas-agent` when an agent is useful.
