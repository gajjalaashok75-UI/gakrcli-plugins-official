# Use Cold Email

Use this command when you want GAKRCLI to apply the `cold-email` skill.

## Usage

```text
/cold-email:use
```

## What It Does

- Loads the `cold-email` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `cold-email-agent` when an agent is useful.
