# Use Cloudflare Workers Expert

Use this command when you want GAKRCLI to apply the `cloudflare-workers-expert` skill.

## Usage

```text
/cloudflare-workers-expert:use
```

## What It Does

- Loads the `cloudflare-workers-expert` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `cloudflare-workers-expert-agent` when an agent is useful.
