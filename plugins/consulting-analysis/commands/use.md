# Use Consulting Analysis

Use this command when you want GAKRCLI to apply the `consulting-analysis` skill.

## Usage

```text
/consulting-analysis:use
```

## What It Does

- Loads the `consulting-analysis` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `consulting-analysis-agent` when an agent is useful.
