# Use Nosql Expert

Use this command when you want GAKRCLI to apply the `nosql-expert` skill.

## Usage

```text
/nosql-expert:use
```

## What It Does

- Loads the `nosql-expert` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `nosql-expert-agent` when an agent is useful.
