# Use Graphify Windows

Use this command when you want GAKRCLI to apply the `graphify-windows` skill.

## Usage

```text
/graphify-windows:use
```

## What It Does

- Loads the `graphify-windows` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `graphify-windows-agent` when an agent is useful.
