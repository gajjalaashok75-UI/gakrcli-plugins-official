# Use Article Writing

Use this command when you want GAKRCLI to apply the `article-writing` skill.

## Usage

```text
/article-writing:use
```

## What It Does

- Loads the `article-writing` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `article-writing-agent` when an agent is useful.
