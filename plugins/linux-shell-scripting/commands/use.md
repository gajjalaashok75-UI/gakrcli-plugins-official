# Use Linux Shell Scripting

Use this command when you want GAKRCLI to apply the `linux-shell-scripting` skill.

## Usage

```text
/linux-shell-scripting:use
```

## What It Does

- Loads the `linux-shell-scripting` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `linux-shell-scripting-agent` when an agent is useful.
