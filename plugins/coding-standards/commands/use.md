# Use Coding Standards

Use this command when you want GAKRCLI to apply the `coding-standards` skill.

## Usage

```text
/coding-standards:use
```

## What It Does

- Loads the `coding-standards` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `coding-standards-agent` when an agent is useful.
