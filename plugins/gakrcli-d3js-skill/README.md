# Gakrcli D3JS Skill Plugin

This skill provides guidance for creating sophisticated, interactive data visualisations using d3.js.

## Overview

This plugin packages the `gakrcli-d3js-skill` skill from the souls library as a first-class GAKRCLI marketplace plugin.

## Installation

```bash
/plugin install gakrcli-d3js-skill@gakrcli-plugins-official
```

## Usage

### Command

```text
/gakrcli-d3js-skill:use
```

### Agent

Ask GAKRCLI to launch `gakrcli-d3js-skill-agent` when you want focused help with this skill's workflow.

### Skill

The skill is available at:

```text
skills/gakrcli-d3js-skill/SKILL.md
```
