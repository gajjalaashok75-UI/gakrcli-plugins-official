# Use Gakrcli D3JS Skill

Use this command when you want GAKRCLI to apply the `gakrcli-d3js-skill` skill.

## Usage

```text
/gakrcli-d3js-skill:use
```

## What It Does

- Loads the `gakrcli-d3js-skill` skill instructions.
- Uses the bundled references, scripts, or assets when the skill includes them.
- Routes focused work to `gakrcli-d3js-skill-agent` when an agent is useful.
